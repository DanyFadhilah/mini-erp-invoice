# Mini ERP Invoice

Frontend application for Mini ERP Invoice built with Next.js, TypeScript, Tailwind CSS, React Query, and Shadcn UI.

## Features

- Login Authentication
- Dashboard
- Customer CRUD
- Product CRUD
- Invoice CRUD
- Invoice Status
- Responsive Layout
- Dashboard Analytics
- Form Validation using Zod

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- TanStack Query
- React Hook Form
- Zod
- Axios

## Installation

```bash
git clone <repository>

cd frontend

npm install
```

## Environment

Create `.env.local`

```env
NEXT_PUBLIC_API_URL=http://localhost:3000
```

## Run

```bash
npm run dev
```
